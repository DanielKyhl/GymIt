# GymIt

A mobile app built with React Native, Expo, and TypeScript. Currently in development.

## What it does

Lets users create an account and log in to track their workouts. The authentication flow is almost complete, with plans to add workout logging, history, and progress tracking.

## Built with

- React Native
- Expo
- TypeScript

## Getting started

1. Install dependencies: `npm install`
2. Start the app: `npx expo start`
